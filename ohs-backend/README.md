# 오행시 백엔드 프로젝트

## 설명

오행시의 Nestjs 기반 REST API 서버입니다.

## 설치

```bash
$ npm install
```

## 실행

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 테스트

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
